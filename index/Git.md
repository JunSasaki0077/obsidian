#Git #IT 

## Gitとは


一言でいうと`ファイルのバージョン管理が簡単にできるツール`です。

**誰が、どのファイルに、何を書き込んだのか、何を削除したのか、
編集前とどう変わったのか**を履歴として残しておき、
必要であればそのバージョンまで戻ることもできる。

様々なコマンドがあり、そのコマンドを用いて
バージョン管理を行う

## よく使うコマンド

#### 確認系
- git status
	- 今の作業状態を確認する
	- 変更したファイルが / [[add]]済か未[[add]]かがわかる
- git log
	- [[commit]]履歴を見ることができる
	- 誰が・いつ・何をしたか確認
- git diff
	- 変更内容の差分を見る
	- [[commit]]前のセルフレビューに必須

#### [[ブランチ]]操作
- git branch
	- ブランチ一覧を見る
	- 今どのブランチにいるか確認
- git branch ブランチ名
	- 新しいブランチを作る
- git switch ブランチ名
	- ブランチを切り替える
- git switch -c ブランチ名
	- 新しいブランチの作成 + 切り替えを同時に行う

#### リモート連携系
- git fetch
	- [[リモート]]の最新情報を取得
	- 安全確認ように使用
- git pull
	- 最新情報を取得 + 反映を一気に行う
	- 便利だけど事故りやすいので使わない
- git push
	- ローカルの[[commit]]を[[リモート]]へ送信
