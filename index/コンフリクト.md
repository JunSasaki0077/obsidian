#Git #エラー 

[[Git]]で起こる現象の一つ
別の人が同じ箇所を上書きしようとした際に
Gitが`どっちを採用すればいいかわからない！`となる状態のこと

main branch
```js
let price = 100;
```

自分の branchで変更
```js
let price = 200;
```

その間にmainも変更された
```js
let price = 120;
```

この状態で現在の[[branch]]を[[merge]]しようとすると
Gitが自動で決められず、結果下記になる。

```js
<<<<<<< HEAD
let price = 150;
=======
let price = 120;
>>>>>>> feature

```
